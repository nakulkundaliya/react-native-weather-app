REACT NATIVE WEATHER APP

### Setup Project

1. Clone Porject from github

   `$ git clone https://github.com/nakulkundaliya/react-native-weather-app.git`

2. Install node_module

   `$ npm i`

3. Run in Android

   `$ react-native run-android`

4. Run in iOS

   `$ react-native run-ios`
